# Import Items Images

You can import images of existing items in bulk using the Import Items Images option in the items list page. Here’s how:

  * Go to the **Items** module under _Items_ from the left sidebar.

  * Click the three dot icon on the top right corner of the page.

  * Select **Import** and choose **Import Item Images.**

  * Click **Choose File** and select the ZIP file containing item images. The ZIP file:

    * Should have a separate folder for each item with images and the folder name should be the item’s SKU.
    * Contains images in PNG, JPEG, or JPG file formats with an individual image not exceeding 5 MB.
    * Maximum file size can be 100 MB.
    * Can contain up to 10 images per folder.
  * Select **Add New Images** if you are adding images for the first time to an item or if you wish to add the new images to the existing ones.

  * Select **Overwrite Existing Images** if you want to completely remove the existing images and add the new ones.

  * Click **Import**.
