# Hallo! Wie können wir Ihnen helfen?

# Notizen

Ihr Zoho-Postfach ist in die Anwendung Zoho Notes integriert. Diese Anwendung ist eine einfache Möglichkeit sicherzustellen, dass Ihre spontanen Ideen nicht verloren gehen. Mit Optionen zum Einbetten von Bildern und Anhängen bietet Notes eine leicht zu visualisierende Benutzeroberfläche.

Inhaltsverzeichnis

  * Meine Notizen
  * Gruppennotizen



Die Anwendung Notes kann über den linken Bereich in Ihrem Zoho-Postfach aufgerufen werden.

##  Meine Notizen

Sie können individuelle Notizen erstellen, um Ihre Ideen zu notieren, Inhalte zu entwerfen, Informationstexte und Zitate von Websites festzuhalten usw. Sie können relevante Bilder in Ihre Notizen einbetten, damit sie auf den ersten Blick aussagekräftig sind.

Im Vorschaubereich von Notes können Sie Anhänge wie Bilder, Audioclips und Dateien zu Ihrer Notiz hinzufügen. Darüber hinaus können diese Notizen zusammen mit den eingebetteten Bildern und Anhängen direkt als E-Mail versendet werden.

Beim Öffnen einer bestimmten Notiz können z. B. Optionen zum Kommentieren der Notiz und Hinzufügen von Eingeladenen angezeigt werden.

​

Sie können auch Erinnerungen für alle Notizen direkt in der **List view** oder **Grid** **view** hinzufügen, indem Sie auf das **Reminder icon** tippen, das eingeblendet wird, wenn Sie den Mauszeiger über eine Notiz bewegen. Um mehr über den Einrichtungsprozess für Erinnerungen zu erfahren, [klicken Sie hier](/mail/help/reminders.html).

##  Gruppennotizen

Sie können Notizen direkt unter Gruppen erstellen. Notizen, die zu Streams hinzugefügt wurden, können direkt in dem entsprechenden Bereich für Gruppennotizen gepostet werden.

Nachdem Sie eine Gruppennotiz erstellt haben, wird die Gruppe benachrichtigt, wenn Sie Änderungen an der Notiz vornehmen. Gruppennotizen, die von anderen Personen als Ihnen selbst erstellt wurden, können nicht bearbeitet werden. Sie können nur als Favorit markiert, gekennzeichnet, als E-Mail versendet oder kommentiert werden.
