# Admin Implementation

Checklist

  * Enable [visitor tracking](https://www.zoho.com/salesiq/help/getting-started-add-welcome-page-code-to-website.html) and [live chat](https://www.zoho.com/salesiq/help/getting-started-add-welcome-page-code-to-website.html) on your website
  * Customize the [chat widget](https://www.zoho.com/salesiq/help/chat-widget-types.html) and [window](https://www.zoho.com/salesiq/help/customize-chat-window.html) according to the website design
  * Set-up your company profile and [business hours](https://www.zoho.com/salesiq/help/settings-company-business-hours.html)
  * Invite [operators](https://www.zoho.com/salesiq/help/settings-operators.html) (colleagues) to assist visitors
  * Add [departments](https://www.zoho.com/salesiq/help/settings-departments.html) (segments) to categorize the operators
  * [ Schedule ](https://www.zoho.com/salesiq/help/visitor-history-module.html)visitor list to the sales teams
  * Schedule visitor, operator, and website [reports](https://www.zoho.com/salesiq/help/live-chat-support-reporting.html)
  * Configure your [portal settings](https://www.zoho.com/salesiq/help/portal-settings-chat-window.html) and emails
  * Deploy [Chatbots](https://www.zoho.com/salesiq/help/developer-guides/bot-introduction.html) to offload the operators
  * [Score](https://www.zoho.com/salesiq/help/visitorhistory-leadscoring.html) the visitor's activities on the site
  * Automate [visitor engagement](https://www.zoho.com/salesiq/help/automation-intelligent-triggers.html) on the website
  * Automate routing [visitors](https://www.zoho.com/salesiq/help/automation-visitor-routing.html) and [chats](https://www.zoho.com/salesiq/help/automation-chat-routing.html) to operators Include
  * Add [Knowledge base](https://www.zoho.com/salesiq/help/articles-templates.html) to live chat window
  * Using any other third-party software then integrate them with SalesIQ
  * Add SalesIQ - [Mobilisten ](https://www.zoho.com/salesiq/in-app-chat-support.html) to your mobile application
