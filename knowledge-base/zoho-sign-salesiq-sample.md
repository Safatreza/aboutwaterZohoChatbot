# Zoho Sign & SalesIQ Kurzreferenz

## Zoho Sign - Digitale Unterschriften

### Dokument zur Unterschrift senden

**Schritt-für-Schritt:**

1. **Zoho Sign öffnen:** https://sign.zoho.eu
2. **Neues Dokument:** "+ Dokument hochladen"
3. **Datei wählen:** PDF, Word, Excel
4. **Empfänger hinzufügen:**
   - Name und E-Mail
   - Rolle: Unterzeichner, CC, Genehmiger
5. **Unterschriftsfelder platzieren:**
   - Drag & Drop auf PDF
   - Typen: Unterschrift, Initialen, Datum, Text
6. **Senden:** E-Mail geht an Empfänger
7. **Tracking:** Status in Echtzeit

**Unterschrift-Workflow:**
- Empfänger erhält E-Mail
- Klickt Link → Dokument öffnet sich
- Unterschreibt direkt im Browser
- Beide Parteien erhalten signiertes PDF

**Integration mit CRM:**
- Aus Zoho CRM → Dokument zur Unterschrift senden
- Signiertes Dokument wird in CRM gespeichert

## Zoho SalesIQ - Live Chat & Chatbots

### Was ist SalesIQ?

Zoho SalesIQ ist die Chat- und Besucherverfolgungsplattform von Zoho.

**Hauptfunktionen:**
- Live Chat für Website
- Chatbots (Zobot)
- Besuchertracking
- Lead-Scoring
- Chat-Routing

### Chat Widget einbinden

**Auf Website:**

1. SalesIQ → Settings → Installation
2. Embed Code kopieren
3. Vor `</body>` Tag einfügen
4. Widget erscheint auf Website

**Anpassung:**
- Settings → Widget → Design
- Farben, Position, Text anpassen

### Zobot erstellen

**Einfacher Bot:**

1. Settings → Bots → Zobot
2. "+ Add Bot"
3. Codeless Bot Builder
4. Flowchart erstellen:
   - Welcome Message
   - Question (collect info)
   - Answer / Action
   - Handoff to human

**Dieser aboutwater Zoho-Assistant ist ein Zobot!**

### Live Chat nutzen

**Als Operator:**

1. SalesIQ öffnen
2. "Visitors" Tab → Aktive Besucher
3. Chat starten oder warten auf Kundenanfrage
4. Chatten
5. Notizen für CRM-Sync

**Chat-Routing:**
- Automatische Zuweisung nach:
  - Department
  - Sprache
  - Operator-Verfügbarkeit
